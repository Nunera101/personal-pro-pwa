## Backend Node opcional

O PWA publicado no GitHub Pages funciona em modo estatico, com dados locais no navegador. Para upload real de video, chat em tempo real e persistencia compartilhada entre aparelhos, publique o backend Node deste mesmo projeto em uma hospedagem separada e vazia.

Nao reutilize ambientes, tokens ou projetos de outros apps. Crie um projeto novo para este app antes de configurar qualquer deploy de backend.
